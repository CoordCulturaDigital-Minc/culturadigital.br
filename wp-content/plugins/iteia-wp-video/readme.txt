Exibe vídeos do iteia através de shortcodes
== Description ==
Este plugin foi desenvolvido para exibir vídeos cadastrados no portal iteia.org.br. O iteia possui um conversor de vídeo que gera arquivos FLV. Este plugin utiliza o player FlowPlayer para exibição dos vídeos.
== Installation ==
1. Faça Upload do plugin `iteia-wp-video.php` para o diretório `/wp-content/plugins/`
1. Ative o plugin no menu de 'Plugins' do WordPress.
1. Pegue o código na página do vídeo no iteia Ex.`http://www.iteia.org.br/videos/seu-tatao` e cole  no lugar do 'vídeocode'
1. Adicione o código `[iteia id=videocode]` no editor de posts.
1. É possível definir largura e altura em pixels, W= largura em pixels e H=altura em pixels. Ex. [iteia id=videocode w=450 h=380]
