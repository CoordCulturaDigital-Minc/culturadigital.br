A theme ready for translation

The theme is now available in French and English, to activate the language you have to go into wp-config.php in the root of your blog on your server and add this: define ('WPLANG', ' en_EN ');

No need logo

Indeed, the theme includes fonts that allow Google to have the text that appears rather elegantly.

Native support of WP-pagenavi

If you want to have a nice navigation on the bottom of your pages just install WP-pagenavi
the theme is ready for this plugin

Choice of color and the background image

The background color is fully customizable, you can even choose a background image and put it through the interface of the theme in the wordpress dashboard.

Integration thumbnails on the homepage

Like many issues in the minds of magazine themes, Codium can integrate a thumbnail (optional) from the pictures of each section, a simple way to give a bit of mirth to your home page. Moreover, this feature only works if necks are in a higher version to wordpress 2.9. If this happens you should know that to regenerate thumbnails of previous articles you can use the excellent plugin auto post thumbnail that will do the work for you by taking the first image as a thumbnail!

Un thème prêt à la traduction

Le thème est aujourd'hui disponible en français et en anglais, pour activer la la langue il faut aller dans wp-config.php situé à la racine de votre blog sur votre serveur et d'ajouter cela : define ('WPLANG', 'fr_FR');

Pas besoin de logo

En effet le thème intègre les polices Google qui permettent d'avoir du texte qui s'affiche plutôt élégamment.

Support de wp-pagenavi

Si vous souhaitez un système de navigation sympathique, installez le plugin wp-pagenavi, le thème le supporte nativement.

Choix de la couleur et de l'image du fond

La couleur du fond est entièrement customisable, vous pouvez même choisir une image de fond et la mettre via l'interface du thème dans le tableau de bord de wordpress.

Intégration des vignettes sur la page d'accueil

Comme beaucoup de thèmes dans l'esprit des thèmes magazines, Codium permet d'intégrer une vignette (au choix) issue des images de chaque article, un moyen très simple pour donner un peu de gaité à votre page d'accueil. Par ailleurs cette fonctionnalité ne fonctionne que si cous êtes dans une version supérieure à wordpress 2.9. Si tel est le cas sachez que pour regénérer les vignettes des articles précédents vous pouvez utiliser l'excellent plugin auto post thumbnail qui va faire le travail pour vous en prenant la première image comme vignette!
